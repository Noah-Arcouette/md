# MD 

MD is a MakeDown file reader with:
- [x] Header styles
- [x] List support
- [ ] check list support
- [ ] under lights ( `========` )
- [x] highlighting support I.E
```
	*Hello, world!*
	`Hello, world!`
	***Hello, world!***
```
- [ ] basic HTML support

- [ ] Style creator

- [x] Linux support
- [x] Cygwin support
- [ ] Windows support

# Install

## Cygwin
```
$ make mk

$ make cyg
```

## Linux
```
$ make mk

$ make linux
```
