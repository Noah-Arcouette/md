# MD 

MD is a MakeDown file reader with:
- [x] Header styles
- [x] List support
- [ ] check list support
- [ ] highlighting support I.E
	```
	*...*
	***...***
	`...`
	```
- [ ] basic HTML support

- [ ] Style creator

- [ ] Linux support
- [x] Cygwin support
- [ ] Windows support

# Install

## Cygwin
```
$ make mk

$ make cyg
```

## Linux
```
$ make mk

$ make linux
```
