# MD 

MD is a MakeDown file reader with:
- [x] Header styles
- [x] List support
- [x] check list support
- [x] under lines ( `========` )
- [x] highlighting support I.E
```
	*Hello, world!*
	`Hello, world!`
	***Hello, world!***
```
- [ ] basic HTML support

- [ ] Style creator

- [x] Linux support
- [x] Cygwin support

# Install

## Cygwin
```
$ make mk

$ make cyg
```

## Linux
```
$ make mk

$ make linux
```

## AUR

[MD](https://aur.archlinux.org/packages/md-git/) also is in the AUR at https://aur.archlinux.org/packages/md-git/
